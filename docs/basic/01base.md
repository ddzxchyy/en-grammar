# 基本句型

## 五种单句

必须符合五种句型才是正确的单句

1. S + V    主语 + 不及物动词
2. S + V + O    主语 + 及物动词 + 宾语
3. S + V + C    主语 + 系动词 + 补语
4. S + V + O + O    主语 + 动词 + 间接宾语 + 直接宾语
5. S + V + O + C    主语 + 动词 + 宾语 + 补语

句型 3 用补语说明主语  
句型 5 用补语说明宾语  

基本句型分为五种，是因为有五种特性不同的动词造成的。
在所有的英语动词中，只有解释为“是”的动词是空的，完全没有意义。
也只有这种动词才需要补语来补足句子的意思。

### 完整的句子

一个完整的句子，必须能够表达完整的意思。这需要以二个部分来完成： 主语和动词。

* 主语：是这个句子所叙述的对象
* 动词：叙述的主要内容
